# OpenCoreSrc
OpenCoreSrc is a 3D game engine currently in active development.  It supports rendering, physics, lighting, and level management.  This repository contains builds and architecture notes.

Inspired by classic id Tech architecture (Quake-like structure), 
but **completely rewritten from scratch** in C with modern approaches.

**This is NOT a Quake engine fork** - it's an independent implementation 
with different file formats, rendering pipeline, and codebase.
